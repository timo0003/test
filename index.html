Introduction à Git

Git est un système de gestion de versions distribué utilisé principalement pour le suivi des modifications dans le code source lors du développement logiciel.

Les Commandes de Base de Git

  • git init : Initialise un nouveau dépôt Git dans le répertoire courant.
  • git clone <url> : Clone un dépôt distant dans un nouveau répertoire.
  • git add <fichier> : Ajoute les modifications du fichier spécifié à l'index (stage).
  • git commit -m "message" : Enregistre les modifications mises en scène dans l'historique du dépôt avec un message descriptif.
  • git push : Envoie les modifications locales vers le dépôt distant.
  • git pull : Récupère les modifications du dépôt distant et les fusionne avec les fichiers locaux.

Étapes pour Configurer un Dépôt Git

  1. Créer un répertoire de projet.
  2. Initialiser le dépôt avec git init.
  3. Ajouter des fichiers avec git add.
  4. Enregistrer les changements avec git commit.
  5. Associer un dépôt distant avec git remote add origin <url>.
  6. Pousser les changements avec git push -u origin master.
